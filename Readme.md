# ⚡ EV-CHATBOT

> **Electric Vehicle Troubleshooting Chatbot** — an experimental project built to test the capabilities of GPT-based models in diagnosing and troubleshooting issues related to electric vehicles (EVs).

This repository demonstrates how conversational AI can be integrated with EV-specific knowledge to provide intelligent, safe, and contextual troubleshooting guidance for EV users and technicians.

## Key Features
- Conversational troubleshooting for EV systems (battery, charging, motor, BMS, etc.)
- LLM-powered responses using GPT-style models
- Context-aware conversations with memory retention
- Modular backend for model and API configuration
- Safety-first — avoids unsafe or technical instructions
- Extensible with domain-specific rules, knowledge bases, and embeddings
- Feedback and session management endpoints for continuous improvement
